# 🚀 Aplikasi Peminjaman Alat Sekolah

---

## 📖 Deskripsi

Aplikasi Peminjaman Alat Sekolah adalah sistem berbasis web yang dirancang untuk mempermudah siswa dalam melakukan peminjaman alat atau fasilitas yang tersedia di sekolah.

Melalui aplikasi ini, proses peminjaman menjadi lebih **efektif, terorganisir, dan transparan**, serta memudahkan admin dalam mengelola data peminjaman, pengembalian, dan ketersediaan alat.

---

## 🎯 Tujuan

* Mempermudah siswa dalam melakukan peminjaman alat
* Membantu admin dalam mengelola data alat dan peminjaman
* Menyediakan sistem monitoring status peminjaman
* Meningkatkan efisiensi pengelolaan sarana sekolah

## 🔄 Alur Sistem

### 🧑‍🎓 Siswa

1. Login ke sistem
2. Melihat daftar alat yang tersedia
3. Mengajukan peminjaman alat
4. Menunggu persetujuan admin
5. Mengembalikan alat sesuai waktu
6. Melihat status dan histori peminjaman

---

### 👨‍💼 Admin

1. Login ke sistem
2. CRUD USER
3. CRUD Kategori
4. Melihat Log_aktivitas
5. Mengelola data alat
6. Melihat permintaan peminjaman
7. Menyetujui / menolak peminjaman
8. Mengatur status (Dipinjam / Dikembalikan)
9. Mengelola laporan peminjaman

---

👨‍💼 Petugas
1. Login ke sistem
2. Mengelola data alat
3. Melihat permintaan peminjaman
4. Menyetujui / menolak peminjaman
5. Mengatur status (Dipinjam / Dikembalikan)
6. Mengelola laporan peminjaman
